  • It was used to give beauty to elements.
  • The current version of CSS is :3.

TYPES OF CSS

  1. Inline css
  2. Internal css
  3. External css

INLINE CSS

  • It will stay within the line,It should be write inside the body part.
  • (h1 style="color:blue") to change text color.
  • (h1 style="color:blue;background-color:black") used to add background color.

INTERNAL CSS

  • It will stay within the file,It should be write inside the head part.
  • It's also will contain same properties like another two.

EXTERNAL CSS

  • It will stay outside the file,It will be write in head part.
  • (link rel="stylesheet" href="./style.css").

CSS SELECTORS

  • Tag selector
  • Class selector
  • id selector
  • Universal selector

CSS BOX MODEL

  • It was all about creating box with the properties.
  • The properties are color,background-color,width,height,border,text-align,padding,margin.
